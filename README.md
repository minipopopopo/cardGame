
# og:supa ⚡️

Og:supa是一个可以i定制Og:image的生成器，允许您为您的帖子生成Og:images，从而使空链接预览成为过去，Og:images用于在社交媒体中显示预览缩略图

## Features

- 🎉 空链接预览已成为过去!
- 🎨 生成可自定义的og：实时图像
- 🚀 保存许多不同的项目型式以供编辑
